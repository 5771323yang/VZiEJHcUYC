# 旅游信息推荐管理系统 java571

## 项目概述

旅游信息推荐管理系统是一款基于前后端分离架构的信息管理系统。该系统致力于为用户提供全面的旅游信息，包括旅游新闻、景区信息、美食信息以及旅游线路等，同时提供完整的管理功能，方便管理员进行内容管理与系统维护。

## 技术栈

- 前端：Vue
- 后端：Java + Spring Boot
- 数据库：MySQL
- 中间件：MyBatis

## 功能模块

### 前台功能

- 展示旅游新闻
- 景区信息展示
- 美食信息展示
- 旅游线路展示
- 在线留言功能

### 后台管理功能

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游线路管理
- 订单信息管理
- 新闻管理
- 系统管理

### 用户功能

- 旅游线路查询与选择
- 订单信息查看

## 系统角色

- 管理员：负责系统的管理与维护
- 普通用户：查询旅游信息，预订旅游线路

## 运行环境

- 前端：现代浏览器
- 后端：Java 8+，Spring Boot 2.x
- 数据库：MySQL 5.7+

## 部署步骤

（部署步骤将详述于配套文档中）

1. 环境准备：Java, MySQL, Maven等
2. 数据库搭建：执行SQL脚本
3. 后端服务部署：编译打包，部署到服务器
4. 前端服务部署：编译打包，部署到Web服务器
